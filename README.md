# Dashboard Interativo: Execuções de Condenados por Assassinato nos EUA

![Power BI](https://img.shields.io/badge/Power%20BI-F2C811?style=for-the-badge&logo=powerbi&logoColor=black)
![Power Query](https://img.shields.io/badge/Power%20Query-217346?style=for-the-badge&logo=microsoft&logoColor=white)

Dashboard analítico desenvolvido no Power BI para exploração e visualização de dados históricos sobre execuções de pessoas condenadas por assassinato nos Estados Unidos.

---

## O Desafio

Analisar um conjunto de dados histórico sobre execuções nos EUA e transformá-lo em um dashboard claro e informativo, permitindo identificar padrões demográficos, temporais e operacionais relacionados à pena de morte.

---

## A Solução

Construção de um dashboard interativo no Power BI com foco em análise exploratória e leitura rápida de indicadores-chave, utilizando Power Query para tratamento básico dos dados e organização do modelo.

Principais análises apresentadas:

- Distribuição de execuções por sexo
- Execuções por faixa etária
- Evolução anual das execuções
- Raças mais executadas
- Método de execução mais utilizado
- Total histórico de execuções
- Identificação do criminoso com maior número de vítimas

---

## Tratamento de Dados (Power Query)

As seguintes etapas de tratamento foram aplicadas:

- Ajuste de tipos de dados (datas, números e textos)
- Padronização de valores categóricos
- Remoção de registros inconsistentes ou incompletos
- Preparação dos dados para uso direto em visualizações e KPIs

> O foco do projeto foi a visualização e análise, com tratamento de dados simples, porém suficiente para garantir consistência nas análises.

---

## Visualizações do Dashboard

Elementos visuais incluídos:

- Gráfico de rosca: distribuição por sexo
- Gráfico de barras: execuções por faixa etária
- Gráfico de barras: execuções por raça
- Gráfico de área: evolução anual das execuções
- Cards de KPI:
  - Total de execuções
  - Ano com mais execuções
  - Método de execução mais utilizado
  - Criminoso com maior número de vítimas

Filtros disponíveis:

- Método de execução

---

## Insights & Análises

O dashboard permite identificar:

- Predominância de execuções de indivíduos do sexo masculino
- Concentração das execuções em determinadas faixas etárias
- Tendência histórica de crescimento e posterior queda no número anual de execuções
- Diferenças significativas entre grupos raciais
- Predominância da injeção letal como método de execução

1. Faça o download da base de dados original

3. Abra o Editor do Power Query
4. Edite a primeira etapa de edição (Fonte) e selecione o caminho para a base baixada
